HTML: Hyper Text Markup Language

  • Creates the structure of the webpage and is made up of elements that work together to provide content and basic framework.
  • Elements types include text, images, header, lists, page divisions, meta related, and semantic.
  • Attributes are inside open tags and also defined in beginning element.
  • The head element contains information about the webpage.
  • The head element is the opening element and contains metadata. Metadata is data about the webpage that is not seen by the user. It contains information about accessibility, search engines, and performance.
  • The head element contains title bar and language setting.
  • The body element represents the visible content shown to the user.
  • The body element has two different types of elements: Semantic and non-semantic.
  • Semantic elements are perferred and important because they make your HTML easier to read by other developers, browser, and accessibility to a wide variety of users.
  • Non-semantic elements are a simple solution to lay out problems. Example: div to create a division in page or span to create an inline division.
  • Headers elements are numbered 1-6, which refers to the level of the heading.
  • The paragraph element wraps text and the letter p in arrows is used.

CCS: Cascading

  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet
  • Inline CSS style sheet: adds style an element directly in the HTML file. inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only.
  • Inline CSS style sheet are useful for quick and permanent changes but are less flexible than external and internal style sheets, because each inline style must be edited individually whenever you make a design change.
  • Internal style sheet contains CSS rules for the webpage in the head section of the HTML file.The rules only apply to that page, and you can configure CSS classes and IDs, which are used to identify elements, to style multiple elements in the page code.
  • In Internal style sheets, a single change to the CSS rule will apply to all tagged page elements.
  • External CSS style sheets are preferreed and keeps CSS rules in separate file
  • Declarations contain two important components: the CSS property we want to apply and the value of the property

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git add -A: add changed files into your working directory to the staging area of git
  • git commit -m "comment": commits changes to your repo. -m flag refers to a message attached.
  • git pull origin main: syncs local branch with the remote branch on Github
  • git push origin "branch": pushes changes to github

  • Control flow is the order in which a computer executes code in a script. In JavaScript, code runs from the first statement at the top of the page, down to the last statement at the bottom of the page. The order of code affects the outcome.
  • Control flow can be interrupted, however. We can use conditional statements and loops to control the order in which code is executed, or if it is even executed at all.
  • One type of conditional statements is an IF statenebt. An if statement is a way of introducing decision-making into our code. When using the if statement, code will only execute if the statement meets a condition, or is true.
  • When the computer encounters an if statement in a JavaScript file, it will check to see if the condition that we provide is truthy. If it is, then it will do whatever action we ask it to do in between the curly brackets. If the condition is not true or has a value that is not truthy, then the computer will ignore whatever is in between the curly brackets and move on.
  • ELSE IF: The computer will continue through all of the conditions that we list, evaluating whether or not a condition is truthy. Once it finds a condition that is truthy, it will run the code in the curly brackets associated with that condition, and then break out of the if statement.
  • if "===" in each condition. The strict equality operator (===) checks to see if two values are equal, and returns a Boolean result true if the values are equal and false if the values are not equal.
  • An ARRAY is a group of related data in a variable and use a for loop to perform a task on each piece of the data, using just a few lines of code.
  • Arrays are especially useful when we want to perform a task on multiple pieces of data at once.
  • Arrays are not limited to contain only a single type of data. A single array can store different types of data, such as strings, numbers, and so on.
  • To work with individual data items inside an array, we need to have one more piece of information: the location of the data item in the array. This location is identified with a unique number called an index. Using each piece of data's index, or location number, we can access individual data inside the array.
